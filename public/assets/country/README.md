# Banderas de países

Coloca aquí las banderas, una por país, con el nombre exacto de su
identificador (ver `src/data/countryFlags.js`).

No he generado banderas reales (además de que muchas quedarían mal como
imagen genérica, es información fácil de conseguir con licencia libre —
p. ej. flagcdn.com). El sistema de carga automática ya funciona: solo
falta que añadas aquí tus archivos.

## Especificación

- PNG, fondo transparente si es posible.
- Relación horizontal estándar de bandera.
- Resolución recomendada: 256×170 px.
- Nombre de archivo = identificador exacto, ejemplo: `spain.png`.
